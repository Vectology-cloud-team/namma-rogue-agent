- reset behavior,
- seed reproducibility,
- movement,
- wall collision,
- combat,
- item pickup,
- inventory actions,
- food,
- traps,
- dungeon-level changes,
- death,
- amulet acquisition,
- return to surface,
- illegal action rejection,
- replay,
- schema validation.
